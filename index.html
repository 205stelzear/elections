Créer une Élection Partagée

Avertissement

Pour utiliser cette fonctionnalitée, un accès Internet est requis puisque le partage d'élection utilise un serveur pour synchroniser les données entre les différents appareils connectés à une élection partagée.

Des requêtes seront donc envoyées à ce serveur pendant le courant de la session de votes pour ainsi mettre à jour correctement l'élection.

Acceptez vous que les données de votre élection soient partagées avec ce serveur?

Le mot de passe sera aussi partagé avec le serveur et les autres appareils qui rejoindront cette élection : assurez vous de ne pas utiliser de mot de passe confidentiel!

Ceci est un rappel du fonctionnement lors de la session de vote :

Pour passer à un prochain électeur, lorsque l'écran est vert et affiche "Votes enregistrés", vous pouvez appuyer sur la barre d'espace sur votre clavier.
Ceci affichera de plus une notification en haut de l'écran affichant le nombre d'électeurs restant. Cette notification s'affichera à chaque fois que la barre d'espace sera appuyée, même pendant qu'un vote est en cours!
De plus, puisque vous avez un écran tactile, dans ce même écran, vous pouvez appuyer le haut de l'écran (la moitié supérieure) pendant quelques instants pour passer aussi au prochain électeur (particulièrement utile pour les appareils n'ayant pas de claviers de connecté).

Aller aux résultats immédiatement

Cette action ne fera pas de requête supplémentaire au serveur et ira simplement à l'affichage des résultats (demandant le mot de passe s'il fut définit).

Cette action est aussi utile lorsque la connexion au serveur est inaccessible ; Les résultats locals seront alors affichés.

Cette action permet de faire une requête au server pour vérifier si tout les votes sont complété.

Cette action permet de télécharger les données les plus récentes des votes et d'aller aux résultats.

Aller aux résultats et supprimer les données

Cette action permet de télécharger les données les plus récentes des votes et d'aller aux résultats, ainsi que de supprimer les données du serveur.

Aller aux résultats et supprimer les données

La requête contiendra la demande de suppression pour enlever cette élection du serveur immédiatement.

Veuillez prendre note que si vous faites cette action, les autres appareils ne pourront plus télécharger les données les plus récentes et ne vont qu'avoir que leur données actuelles lors de l'affichage de leur résultats.

Ainsi, il est recommandé d'utiliser cette action sous suivant l'une des deux situation:
  • après avoir ouvert les résultats avec tous les autres appareils participant à cette élection
  • seul cet appareil affichera les résultats de cette élection partagée
